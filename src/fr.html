Impossible de scanner mon certificat.

  1. Disposez d'une bonne source de lumière et alignez votre code QR avec le bord de la zone de numérisation. La taille de l'aide visuelle est la taille du code dont nous avons besoin. Ton code doit être à plat (le papier froissé ne fonctionnera pas bien).
  2. Si vous avez une capture d'écran ou une image de votre code Qr, vous pouvez utiliser le bouton "Importer une image".
  3. Veuillez utiliser la version EU de votre certificat. Voir la question suivante si vous ne savez pas où obtenir le vôtre.
  4. Nous supportons le certificat COVID numérique de l'UE de vaccination, de rémission, de test.
  5. Si tout le reste échoue, vous pouvez nous contacter ou ouvrir rapport de bug.

Qu'est-ce que ce projet?

Cette application Web analyse votre certificat COVID numérique européen (vaccin, test ou guérison) et en génère un passbook à partir de ce dernier. Vous pouvez ensuite l'importer dans le wallet de votre iPhone, ou si vous êtes un utilisateur Android, importez-le dans une application qui supporte le format.

Puis-je enregistrer le certificat sur Android ?

Oui ! Vous pouvez enregistrer le fichier .pkpass et l'ouvrir avec une application qui supporte ce format, par exemple :

  • Stocard (Google Play Store)
  • PassAndroid (F-Droid)
  • WalletPasses (Google Play Store)

Vous pouvez également prendre une capture d'écran de l'aperçu du passe que nous affichons avant le téléchargement; il contient toutes les informations pertinentes que vous voyez sur votre certificat papier.

Comment puis-je obtenir mon certificat EU DCC ?

🇧🇪 En Belgique, vous pouvez télécharger votre certificat depuis (français), Mijngezondheid.be (néerlandais) ou Meinegesundheit.be (allemand).

🇬🇷 En Grèce, vous pouvez télécharger votre certificat à partir de Gov.gr ou de myKEPlive.
Note : Assurez-vous de télécharger le certificat EU.

Comment ça marche ?

Lorsque vous scannez le code, nous lisons les données et créons un passbook (le fichier nécessaire pour avoir une carte dans votre portefeuille). Nous envoyons une somme de contrôle pour la signature numérique à notre serveur, sans envoyer vos données. Votre téléphone génère le fichier final et vous permet de le sauvegarder.

Est-ce sécurisé ?

Nous avons déployé beaucoup d'efforts pour le rendre aussi sûr que possible. Vos données personnelles ne quittent jamais votre téléphone. Apple impose une signature numérique pour ouvrir un passe. Pour générer cela, l'application n'envoie qu'une somme de contrôle de vos données à notre serveur, et il est impossible de deviner le contenu original. La dernière étape se produit sur votre téléphone : vous prévisualisez vos informations avant d'enregistrer le passe. Nous ne suivons personne, nous ne recueillons aucune information personnelle, et nous n'utilisons pas de cookies.
